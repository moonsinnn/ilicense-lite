# Security Policy

## Supported Versions

当前主分支（`main`）为受支持版本。历史版本默认不提供安全修复承诺。

## Reporting a Vulnerability

请不要通过公开 issue 报告安全漏洞。

请发送邮件到维护者安全邮箱（请在仓库设置中替换）：`bingbo.zh@gmail.com`，并包含：

- 漏洞类型与影响范围
- 复现步骤（PoC）
- 受影响版本或提交哈希
- 建议修复方式（可选）

## Response Timeline

- 48 小时内确认收到
- 7 天内给出初步评估
- 30 天内给出修复计划或缓解方案

## Disclosure

在补丁发布前，请避免公开披露细节。修复发布后会在 release note 中标注安全修复信息。
