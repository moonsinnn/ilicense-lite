# Code of Conduct

## Our Standards

为保证社区协作效率与安全，参与者应：

- 尊重不同观点与技术背景
- 以事实和复现依据讨论问题
- 接受建设性反馈
- 聚焦问题本身，避免人身攻击

不可接受行为包括：

- 侮辱、歧视、骚扰
- 恶意刷屏、挑衅、威胁
- 未经允许公开他人隐私或敏感信息

## Scope

本行为准则适用于：

- 仓库 issue、PR、讨论区
- 与项目相关的公开或私下沟通

## Enforcement

维护者可对违规行为采取措施，包括：

- 警告
- 暂时限制互动
- 永久封禁

## Reporting

如遇不当行为，请联系维护者（建议在仓库资料中配置专用邮箱）。
